# Raj Singh - Personal Portfolio

A modern, premium personal branding website built with React, Vite, and Tailwind CSS.

## Tech Stack

- **React** - UI Framework
- **Vite** - Build Tool
- **Tailwind CSS** - Styling
- **Lucide React** - Icons

## Getting Started

### Install Dependencies

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Deployment

This project is configured for Vercel deployment. Simply connect your GitHub repository to Vercel and it will auto-deploy.

### Manual Vercel Deployment

```bash
npm install -g vercel
vercel
```

## Project Structure

```
src/
├── components/
│   ├── Navbar.jsx      # Sticky navigation
│   ├── Hero.jsx        # Hero section with CTA
│   ├── Approach.jsx    # Approach highlight card
│   ├── Achievements.jsx # Animated counters
│   ├── About.jsx       # About section
│   ├── Expertise.jsx   # Core expertise cards
│   └── Contact.jsx     # Contact CTA section

├── main.jsx            # Entry point

```
